1. Для блока-обёртки подойдет section; для логотипа и сообщения об ошибке - h1, h2; для списка сервисов - ul;
   для пунктов списка - li; для футера - footer; для ссылок - a;
   для поля ввода и кнопки - input:search и input:submit.

2. Поле ввода, кнопка, ссылки. При наведении на кнопку и ссылки курсор должен меняться,
   также при наведении на ссылки появляется нижнее подчеркивание. Поле ввода реагирует на ввод с клавиатуры.

3. Установить содержащиму элементу обертки основного блока свойства:
   display: flex;
   align-items: center;
   justify-content: center;

4. Установить содержащиму элементу футера и основного блока свойства:
   min-height: 100vh;
   justify-content: space-between;

5. Перенос строки.
